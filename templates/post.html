Python

Python é uma linguagem de programação de alto nível, interpretada por script, imperativa, orientada à objetos, funcional, de tipagem dinâmica e forte. A linguagem foi projetada com a filosofia de enfatizar a importância do esforço do programador sobre o esforço computacional. Prioriza a legibilidade do código, sobre a velocidade ou expressividade

Django | Django Rest

Django é um framework para desenvolvimento rápido para web, escrito em python, e que utiliza o padrão model/template/view(MTV).

Tem como principais caracteristicas:

Mapeamento relacional de objetos; onde definimos a modelagem de dados através de classes em python.

Interface administrativa, formulários, sistema de templates, sistema de cache, internacionalização, etc.


Flask

Flask é um framework web escrito em python. É classificado como microframework porque não requer ferramentas ou bibliotecas particulares, mantendo um núcleo simples, porém, extensível.


Bancos de dados relacionais - SQL

Banco de dados relacional, é percebido pelo usuário como tabelas, ou mais formalmente, relações.

PostgreSQL

PostgreSQL é um sistema gerenciador de banco de dados objeto relacional. Hoje o PostgreSQL é um dos SGBDS (sistema gerenciador de banco de dados) de código aberto mais avançados, contando com recursos como: Consulta complexa, Chave estrangeira, Integridade transacional, Controle de concorrência multi versão, etc.


MySQL

O MySQL é um SGBD(Sistema de gerenciamento de banco de dados), que utiliza a linguagem SQL como interface. É atualmente um dos sistemas de gerenciamento de bancos de dados mais populares no mundo.

Docker

Docker é um conjunto de produtos de plataforma como serviço, que usa virtualização de nível de sistema operacional para entregar softweare em pacotes chamados contêineres. Os contêineres são isolados uns dos outros, e agrupam seus próprios softweares, bibliotecas e arquivos configuração.

Docker compose

Docker compose é uma ferramenta para definir e executar aplicativos docker de vários contêineres. Com o compose, usamos um arquivo YAML para configurar os serviços de nosso aplicativo. O compose funciona bem em todos os ambientes: produção, preparo, desenvolvimento, testes, bem como fluxos de trabalho de CI.


Github

Github é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão. Ele permite que programadores, contribuam em projetos privados e/ou open source de qualquer lugar do mundo. É amplamente utilizado por programadores para divulgação de seus trabalhos ou para que outros programadores contribuam com o projeto, alem de prover fácil comunicação através de recursos que relatam problemas ou misturam repositórios remotos.
